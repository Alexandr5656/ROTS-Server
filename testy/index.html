Instructions

First you have to download the template zip file by clicking the "Download Template" button. After unzipping the folder open the template.py file and edit it. One edit inside the commented lines. After you're done upload the file and wait for it to run. While the robot is looking for a file it will blink red. Once it has found a file the robot's leds should be off and the next file in the queue will run.
